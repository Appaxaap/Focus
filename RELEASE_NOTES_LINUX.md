## First Linux Release

Focus v2.2.4 marks the first official Linux release of Focus.

This release brings the desktop experience to Linux with a keyboard-oriented workflow, native desktop integration, and a self-contained release bundle that is ready to run without additional app packaging steps.

The goal of this release is to make Focus feel fast, practical, and at home on Linux desktops while establishing a solid base for future Linux updates.

## Overview

Focus is an offline Eisenhower Matrix task manager built for users who want a structured way to organize work without accounts, cloud sync, or unnecessary complexity.

With this first Linux launch, Focus is now available as a dedicated desktop build that supports fast task management, clear navigation, and distraction-free use on modern Linux systems.

## Keyboard-First Workflow

Focus on Linux is designed to work well with the keyboard, reducing friction during task capture, navigation, and editing.

Current desktop shortcuts include:

- `Ctrl + K` to open the command palette
- `Ctrl + N` to create a new task
- `Ctrl + F` to toggle Focus Mode
- `Ctrl + H` to toggle completed tasks
- `Ctrl + ,` to open settings
- `Ctrl + /` or `F1` to open shortcut help
- `1 - 4` to select quadrants
- `J / K` or `Up / Down` to move task selection
- `Enter` to open the selected task
- `Space` or `X` to complete the selected task
- `Delete` or `Backspace` to delete the selected task
- `Alt + 1 - 4` to move a selected task between quadrants

Task editor shortcuts include:

- `Ctrl + Enter` to save
- `Esc` to cancel
- `Ctrl + Q` to toggle the priority panel
- `Ctrl + D` to cycle quick dates
- `Ctrl + 1 - 4` to set priority directly
- `Ctrl + Shift + 1 - 3` to assign quick dates directly
- `Tab` to move between fields

## Desktop Experience

This Linux release includes a desktop-focused interface built for larger screens and faster interaction.

Highlights include:

- A dedicated desktop home layout
- Command palette support for faster actions
- Improved task selection and editing flow
- Shortcut help dialog for discoverability
- Desktop settings flyout integrated with the main UI
- Better support for focused, low-friction task management

## Linux Integration

This release also introduces the core Linux packaging and metadata required for a proper desktop app experience.

Included in this release:

- Official Linux application ID
- Desktop entry metadata
- Application icon integration
- Linux release workflow for GitHub builds
- Self-contained release archive for x86_64 systems

The downloadable archive extracts to a complete bundle directory and can be run directly from the extracted folder.
